Paseando Ando

“Paseando Ando” consiste en una aplicación Web que ofrece al usuario posibles destinos de paseos turísticos en las cercanías a un lugar. La idea es que el usuario pueda consultar sitios turísticos para visitar indicando el lugar de origen y destino. La aplicación le ofrece una serie de sitios y le indica varias formas de llegar a ese sitio desde su origen. El usuario selecciona una ruta, la cual se muestra en un mapa. Además, cuando el usuario ingresa el destino, se le muestran algunas fotos del sitio, la calificación que tiene, el horario de apertura y comentarios si existen.
